1. C#

使用c#语言，您可以通过我们公司提供的c#动态库，进行自由开发(坐标控制、角度控制、io控制、夹爪控制等），控制我们公司已经研发出来的部分机器人。
支持的机械臂型号：myCobot280、320和myPalletizer 260。


1.1. C#是什么？

C#是微软公司发布的一种由C和C++衍生出来的面向对象的编程语言、运行于.NET Framework和.NET Core(完全开源，跨平台)之上的高级程序设计语言。
C#与Java有着明显的不同，它借鉴了Delphi的一个特点，与COM（组件对象模型）是直接集成的，而且它是微软公司 .NET windows网络框架的主角。
C#使得C++程序员可以高效的开发程序，且因可调用由 C/C++ 编写的本机原生函数，而绝不损失C/C++原有的强大的功能。因为这种继承关系，C#与C/C++具有极大的相似性，熟悉类似语言的开发者可以很快的转向C#.

适用设备：

  • myCobot 280

    • myCobot 280 M5
    • myCobot 280 for Arduino

  • myCobot 320

    • myCobot 320 M5
